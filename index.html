Data Science Course Guide

Discover Berkeley's many interdisciplinary paths for data science.

Download this project as a .zip file Download this project as a tar.gz file

course-guide

One of the highlights of Berkeley's emerging data science program is its interdisciplinary nature -- however, no specific resource exists for students to find all the data science related classes Berkeley has to offer. This project is an effort to fulfill that need by the Berkeley Institute for Data Science Mapping Research Team.

Vinitra Swamy (Team Lead), Jeff Leong, Sunny Agarwal, Erin Hancock, Nancy Li
